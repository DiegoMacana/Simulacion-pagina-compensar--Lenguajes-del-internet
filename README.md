## 📌 Descripción

Este proyecto consiste en una **simulación de la página principal del entorno virtual de UCompensar**, desarrollada como ejercicio académico para el curso de **Accesibilidad Web**.

La página replica la estructura básica de un entorno virtual educativo, incluyendo navegación, menús desplegables, área del curso y presentación de contenido, utilizando **HTML y CSS**.

## 🖥️ Vista general del proyecto

La página simula el entorno de un curso virtual e incluye:

- Encabezado con **logo institucional**
- **Menú de usuario** con opciones de perfil
- **Sistema de notificaciones**
- **Menú principal de navegación**
- **Ruta de navegación**
- **Panel de contenidos del curso**
- **Sección de presentación**
